# E-Book-Marketplace-App-Using-Flutter-Getx-Firebase
E-Book Marketplace is a Flutter-based application that provides a seamless platform for users to browse eBooks.
## Tech Stack

**Client:** Flutter,Dart

**Server:** Firebase.

## Features

- Pdf Reading View
- Text to Speech
- Add Book
- Delete Book 
- Update Book
- User Management
